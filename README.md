## 快速开始
1. 克隆仓库：`git clone https://github.com/MoNi-2025/blog.git`
2. 安装依赖：`bundle install`
3. 本地预览：`bundle exec jekyll serve`
## 文章发布
1. 创建新分支：`git checkout -b post/new-article`
2. 编写Markdown文件并添加Front Matter
3. 本地预览：`bundle exec jekyll serve`
4. 提交PR并等待审核
